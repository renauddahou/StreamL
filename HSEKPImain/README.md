# HSE-KPI

une application de gestion de base de donnée connectée à postgresql avec interfarce utilisateur (login)

✨ Elle est une application d'analyse et de suivi des indicateurs de performance HSE.

✨ Vous pouvez ajouter; modifier; supprimer et visualiser vos données avec des graphes.

✨ Vous pouvez aussi téléchager vos données selon des intervalles de date.
